# Velaro Tiles - Coming Soon Page

A premium, responsive "Coming Soon" landing page for **Velaro Tiles**, a brand crafting India's finest tiles. 

## 🎨 Design & Aesthetic
The design reflects a "Proudly Indian" luxury theme:
- **Color Palette:** Deep Charcoal/Slate background contrasted with rich Saffron (`#FF9933`) and Metallic Gold (`#D4AF37`) accents.
- **Typography:** An elegant pairing of *Playfair Display* for headings and *Outfit* for modern, clean body text.
- **Micro-Interactions:** Smooth, ambient background tile animations, sleek glassmorphism styling (`backdrop-filter`) for the main card, and subtle hover effects on interactive elements.

## 🚀 Features
- **Responsive Layout:** Works flawlessly across Desktop, Tablet, and Mobile devices.
- **Dynamic Countdown Timer:** Built with Vanilla JavaScript to build anticipation for the launch.
- **Contact Integration:** A direct mailto: link for easy communication before launch.
- **Animated Background:** Generates random, floating tile squares to add depth without distracting from the core message.

## 🛠 Tech Stack
This project is built using strictly native, vanilla web technologies to ensure maximum performance and minimum dependency overhead:
- **HTML5:** Semantic architecture.
- **CSS3:** Custom properties (variables), Flexbox/Grid layouts, Keyframe animations, and Glassmorphism.
- **JavaScript (ES6):** DOM manipulation, background generation logic, and interval-based time calculations.

## 💻 Local Development
To run this project locally, simply clone the repository and open `index.html` in any modern web browser, or serve it using a local development server:

```bash
# Clone the repository
git clone https://github.com/thepixelketchup/velaro-website.git

# Navigate into the directory
cd velaro-website

# Start a local server (requires Node.js)
npx serve .
```

Then visit `http://localhost:3000` in your browser.

## ☁️ Deployment
This static template is perfectly optimized for immediate deployment on platforms like **Vercel**, **Netlify**, or **GitHub Pages** with zero configuration required.
